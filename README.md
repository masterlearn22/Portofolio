## ✨ Features

- **Responsive** — Mobile-first design, works on all screen sizes
- **Typing Effect** — Animated role titles in the hero section
- **Flip Photo** — 3D card flip on profile photo hover
- **Animated Counters** — Stats count up when scrolled into view
- **Scroll Reveal** — Sections fade in as you scroll
- **Project Filter** — Filter projects by category (Backend / Full-Stack / ML)
- **Sticky Navbar** — Shrinks on scroll with active section indicator
- **CV Download** — One-click PDF download from navbar and hero

## 🛠 Tech Stack

`HTML5` · `CSS3` · `JavaScript (ES6+)` · `Google Fonts`

## 📁 Project Structure

```
portofolio/
├── index.html              # Single-page HTML
├── css/
│   └── style.css           # Styles, layout, animations
├── js/
│   ├── main.js             # Navbar, hamburger, smooth scroll
│   ├── animations.js       # Typing effect, scroll reveal, counters
│   └── projects.js         # Project category filter
└── assets/
    ├── images/
    │   ├── profile.jpg     # Front profile photo
    │   └── profile-back.jpg # Back profile photo (flip)
    └── cv/
        └── cv.pdf          # Downloadable CV
```

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/username/portfolio.git
   cd portfolio
   ```

2. **Add your assets**
   - Place your photos in `assets/images/`
   - Place your CV PDF in `assets/cv/`

3. **Open in browser**
   ```bash
   # Simply open index.html, or use a local server:
   npx serve .
   ```

## 📝 Customization

| What | Where |
|---|---|
| Name & bio | `index.html` → Hero & About sections |
| Skills | `index.html` → `.skill-tag` elements |
| Projects | `index.html` → Projects section |
| Experience | `index.html` → Timeline items |
| Contact info | `index.html` → Contact section |
| Colors & fonts | `css/style.css` → `:root` variables |
| Typing roles | `js/animations.js` → `roles` array |
