A modern, responsive, and professional personal portfolio website built to showcase my skills, experience, and projects. Designed with a sleek aesthetic, smooth scrolling animations, and dynamic content filtering.

## ✨ Key Features

- **📱 Fully Responsive**: Optimized for seamless viewing on all devices (mobile, tablet, desktop).
- **🎨 Modern UI/UX**: Clean, dark-themed design with smooth transitions and hover effects.
- **🪄 Scroll Animations**: Elements beautifully fade and slide into view as you scroll using Intersection Observer.
- **🔍 Dynamic Project Filtering**: Easily sort projects by categories such as Backend, Full-Stack, and Machine Learning.
- **📄 CV Download**: Quick access to download my latest professional resume.
- **✨ Interactive Elements**: Typing effects, animated counters, and a sticky navigation bar that adapts to scroll position.

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Design/Styling**: Custom CSS Variables, Flexbox, CSS Grid
- **Typography**: Google Fonts
